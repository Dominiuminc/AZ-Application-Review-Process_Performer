# Objective
Extract all income and asset information from the Income & Asset Questionnaire document, including validation checks to ensure completeness and consistency between claimed items and detailed information provided.

# Instructions

1. **Image Processing**:
   - Recognize and extract text from the document images using OCR (Optical Character Recognition).
2. **Extraction**:
   - Extract information from the document into the JSON format below.

For the validation fields, mark them true or false as follows:
    - "all_page1_fields_are_completed": true if all fields on page 1 are marked either "YES" or "NO"
    - "all_incomes_described": true if all fields marked "YES" on page 1 have at least one corresponding entry under "INCOME SOURCES" on page 2
    - "all_assets_described": true if all fields marked "YES" on page 3 have at least one corresponding entry under "ASSET SOURCES" on page 4
    - "has_disposed_assets_answered": true if field 16 has a checkmark for either "HAVE" or "HAVE NOT"
    - "described_disposed_assets_if_necessary": false if field 16 is marked "HAVE" and there are no assets listed under "If applicable". true otherwise.
