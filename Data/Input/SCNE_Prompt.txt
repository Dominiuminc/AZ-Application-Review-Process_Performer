Extract information from Self Certification of Non-Employment  images to determine conditions and produce a JSON output following the provided schema.

- **Task Overview**: Analyze the provided document images, extract relevant data, and evaluate the specified conditions. Then format the results into the given JSON schema.

## Steps to Complete the Task

1. **Image Processing**:
   - Recognize and extract text from the document images using OCR (Optical Character Recognition).

2. **Data Extraction Criteria**:
   - Confirm answer to employment question is "No"
   - Confirm following information filled as required by instructions
   - Confirm signature and date at the bottom

3. **Logic for Determining Outputs**:
   - **IsEmploymentQuestionAnswered**: This should be `true` if the yes/no employment question is answered.
   - **IsAnsweredNo**: This should be `true` if the employment question is answered "no".
   - **IsFollowUpCompleted**: This should be `true` if follow-up information is filled in when answering no.
   - **HasSignatureAndDate**: This should be `true` if the document has signature and date at the bottom.

## Output Format

The output must adhere to the following JSON schema:

```json
{
  "validation": {
    "employment_question_answered": "boolean",
    "employment_answered_no": "boolean",
    "follow_up_completed": "boolean",
    "employment_is_not_null": "boolean",
    "has_signature": "boolean",
    "has_date": "boolean"
  },
  "employment_section": {
    "employment_question": {
      "yes": "checked/unchecked",
      "no": "checked/unchecked"
    },
    "follow_up_information": {
      "reason_for_non_employment": "string",
      "additional_details": "string"
    }
  },
  "applicant_information": {
    "name": "string",
    "address": "string",
    "phone": "string"
  },
  "signature_section": {
    "signature_present": "boolean",
    "date_signed": "string",
    "printed_name": "string"
  }
}
```
